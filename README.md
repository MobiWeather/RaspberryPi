# センサーデータ計測 & クラウド送信システム

本システムは、Raspberry Pi等のLinuxデバイス上で各種センサー（GPS、SHT41、大気圧、加速度、カメラ）からデータを取得し、ローカルへのCSV保存およびクラウドサービス（Ambient等）へのデータ送信を定期的に行うシステムです。

---

## 1. 動作環境・バージョン

本システムは以下の環境およびバージョンで動作確認を行っています。

* **OS:** Linux (Raspberry Pi OS 64-bit 推奨)
* **Python:** `3.10.x` 以上
* **主要パッケージ・依存関係:**
    * `requests` (クラウドAPI通信用)
    * `libcamera` / `rpicam-apps` (カメラ撮影用コマンド)
    * その他、`sensors` モジュール内で依存するI2C/シリアル通信ライブラリ

---

## 2. GitHub登録時の注意点（重要）

本プロジェクトをGitHubなどの公開リポジトリに登録する際は、**仮想環境のフォルダ（`.venv` や `env`）や、パスワードが記載されたファイル（`.env`）を絶対にコミットしないでください。**

プロジェクトのルートディレクトリ（一番上の階層）に **`.gitignore`** ファイルを作成し、以下の内容を記述して追跡対象から除外します。

### `.gitignore` の設定例
```text
# 1. Pythonの仮想環境フォルダを無視
.venv/
venv/
env/

# 2. 環境変数・機密情報ファイルを無視
.env

# 3. 自動生成されるキャッシュファイルを無視
__pycache__/
*.pyc

# 4. センサーのデータログを無視
log/
